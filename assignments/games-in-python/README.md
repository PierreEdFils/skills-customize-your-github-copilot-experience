
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game in Python that practices string manipulation, loops, conditionals, and user input handling.

## 📝 Tasks

### 🛠️	Build the Hangman Game

#### Description
Implement a command-line Hangman game that selects a secret word and lets the player guess letters until they either reveal the full word or run out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses and reveal correct letters in the word (e.g., _ _ a _ _)
- Track and display remaining incorrect guesses
- End the game when the word is guessed or attempts are exhausted
- Display a clear win or lose message
- Handle repeated guesses without penalizing the player

## Starter Code

A starter scaffold is provided in `starter-code.py` to help you get started. Run it with:

```bash
python3 starter-code.py
```

Optional extensions (not required): add difficulty levels, load words from a file, or display a simple ASCII-art hangman.
